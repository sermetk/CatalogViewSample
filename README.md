# CatalogViewSample
Xamarin Forms Catalog View Sample

# Usage
```
 <custom:CatalogView
        x:Name="catalogView"
        Margin="20"
        ColumnSpacing="5"
        HorizontalOptions="Center"
        ItemSource="{Binding CatalogItemSource}"
        ItemTemplate="{StaticResource ItemTemplate}"
        RowSpacing="10"
        ShapeHeight="140"
        ShapeWidth="80" />
```
